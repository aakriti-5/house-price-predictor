\# 🏠 House Price Predictor



A Machine Learning Model that predicts house prices with \*\*89.5% accuracy\*\* using Random Forest.



\## 📊 Results

\- \*\*Algorithm:\*\* Random Forest Regressor (100 trees)

\- \*\*Accuracy:\*\* 89.5%

\- \*\*Mean Absolute Error:\*\* $19,164

\- \*\*Dataset:\*\* 1,460 houses, 7 features



\## 🔍 Key Findings

\- Overall Quality is the #1 factor (57.9% importance!)

\- Living area matters more than number of bedrooms

\- Bedrooms are the LEAST important feature (only 1.3%)
